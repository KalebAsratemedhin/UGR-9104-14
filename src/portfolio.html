Efoyta Doctor's Appointment Booking Application

100%

This is a group project I did in my third year at AAiT for my Fundamentals of Software Engineering course final project. It was a web application for hospitals and patients to manage appointments online. The web application enables patients to book appointments with the doctor of their choice at their preferred time. We made it in the hope of minimizing patient waiting time at hospitals and bettering the patient experience at hospitals. Doctors also benefit from the app as they can manage their daily visitors.

I had the role of managing and leading our team during this project. As such I participated at every stage of the developement process starting from requirement specification and proposal up to the deployment and final testing. In this project I was able to work with: Biniyam Assefa, Yohannes, Meklit Asrat, Isam and Daniel Demerew.

Micro-mouse and Maze Solving Algorithm

Progress:100%
100%

In the summer of 2023, I saw a video on youtube about the fastest maze solving micro mouse competition. The competition is a worldwide event and is most popular in the UK, U.S., Japan, Singapore, India, South Korea and becoming popular in subcontinent countries such as Sri Lanka. After watching the video, I was convinced that I had to try it out and then I did. For the next two months, I was toiling to make my very own micro-mouse robot and write the maze solving algorithm. I finally did!

Micromouse is a small robotic mouse competing to solve a 16×16 maze. The maze is made up of a 16×16 grid of cells, each 180 mm square with walls 50 mm high. The mice are completely autonomous robots that use infrared sensors to locate and must find their way from a predetermined starting position to the central area of the maze unaided. The mouse needs to keep track of where it is, discover walls as it explores, map out the maze and detect when it has reached the goal. Having reached the goal, the mouse will typically perform additional searches of the maze until it has found an optimal route from the start to the finish. Once the optimal route has been found, the mouse will traverse that route in the shortest achievable time.

Micro-mouse is a challenge of both the hardware and algorithm design. For the maze solving algorithm I used a simple depth-first search (DFS). To apply DFS we first transform our maze into a graph. Individual paths become edges and junctions become nodes. The mouse will start at the top node and work its way down until the exit node. By convention, whenever the mouse faces a choice,it goes to the left. If it runs into a dead end, it backtracks up until a new path opens up to the right. In the best case scenario, our exit node is at the bottom left. In the worst case, the node is at the bottom right and the entire graph will be visited. This could take a long time if the maze is big! DFS is a simple algorithm and it doesn’t take up a lot of space in memory.

For the hardware part, I used the following

  • IR Sensors - Sharp GP2D120
  • Microcontroller - Freescale MC9S12C128
  • Microcontroller module board - EVBPlus DragonFly12
  • Stepper Motors - Shinano Kenshi SST39D1010
  • Power Transistors for motors - Fairchild FDB6030BL
  • DC-DC Converter - Hitachi FPMR12TR7508NA

Fake News Detector AI-System

Progress:10%
10%

This project involves natural language processing (NLP), machine learning, and data preprocessing techniques.Collect a dataset of news articles labeled as either "fake" or "real." Several datasets are available online, such as the FakeNewsNet dataset or the LIAR-PLUS dataset.

The software will perform text preprocessing tasks, including lowercasing, tokenization, and removing stop words and punctuation. Use techniques like stemming or lemmatization to normalize words. It will also convert the text data into numerical features that can be used by machine learning models using TF-IDF (Term Frequency-Inverse Document Frequency) and word embeddings (Word2Vec, GloVe).

I will use a machine learning model suitable for text classification from the common models such as:

Naive Bayes
Simple and effective for text classification tasks
Logistic Regression
Widely used for binary classification.
Support Vector Machines (SVM)
Effective in high-dimensional spaces.

The specific technologies I will use include:

Python
The primary programming language for machine learning and data processing
Scikit-Learn
A powerful library for machine learning in Python.
NLTK (Natural Language Toolkit)
Provides tools for working with human language data.
TensorFlow or PyTorch
Deep learning frameworks for building and training neural network models.
Flask or Django
Web frameworks for creating a user-friendly interface.
